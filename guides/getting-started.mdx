## 🎮 Join SuperAwesome

Det første du skal gøre er at joine serveren.

### Sådan gør du

1. Åbn **Minecraft Java Edition**.
2. Gå til **Multiplayer**.
3. Klik på **Add Server**.
4. Indtast følgende:
   - **Server Name:** `SuperAwesome`
   - **Server Address:** `mc.superawesome.dk`
5. Klik **Done**, og join serveren.

Når du er kommet ind på serveren, kan du skrive:

```mcfunction
/sm help
````

for at se alle de tilgængelige kommandoer.

---

## 🔗 Link din Discord-konto

For at få adgang til et nice server dashboard skal du linke din Discord-konto.

Skriv følgende kommando på serveren:

```mcfunction
/sadiscordlink
```

Du modtager nu et link i chatten.

1. Klik på linket.
2. Log ind på Discord, hvis du ikke allerede er logget ind.
3. Godkend forbindelsen.

✅ Din Discord-konto er nu linked til din Minecraft-konto.
