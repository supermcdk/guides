### Introduktion

Vil du gerne gøre din server mere personlig og få nem adgang til serverens vigtigste indstillinger—uden at skulle rode i filer eller kommandolinjen? Så er **Server Manager** det perfekte værktøj til dig.

Server Manager giver dig en brugervenlig måde at ændre alt fra *servernavn, MOTD, serverikoner, regler, whitelist, gameplay-indstillinger* og meget mere. Du kan med få klik tilpasse din server, så den passer til dig og dine spilleres behov.

> Har du brug for hjælp undervejs, kan du altid spørge i [#support](https://discord.com/channels/739186031412379738/1083887717508718702) på vores Discord.

### Hvad er Server Manager?

**Server Manager** er et værktøj i dit SuperAwesome panel, der gør det super nemt at styre din server uden teknisk kendskab.  
Du kan blandt andet:

- Ændre **MOTD** (den tekst der vises i serverlisten)
- Skifte **serverens ikon**
- Aktivere eller slå **whitelist** til og fra
- Justere **serverindstillinger** såsom difficulty, PvP, max spillere mm.
- Skifte **serverens "flavour"** (f.eks. Survival, SMP, KitPvP)

Alt dette kan gøres med få klik – og ændringerne træder ofte i kraft med det samme.

### Sådan bruger du Server Manager

Vi guider dig her igennem, hvordan du finder og bruger Server Manager til at administrere din server.

1. **Start med at logge ind på dit panel**  
   Gå til dit SuperAwesome serverpanel, og vælg den server, du gerne vil redigere.
   
2. **Åbn Server Manager**  
   I sidebar-menuen finder du "Server Manager". Klik på den for at åbne værktøjet.  
   Her bliver du mødt af et dashboard med forskellige sektioner.

3. **Tilpas din server efter dine ønsker**  
   Du vil nu se en række indstillinger, du nemt kan ændre. De mest populære er:
   
   - **MOTD** → Det, der vises i serverlisten. Gør din server mere indbydende!  
   - **Flavour** → Vælg hvilken type server du gerne vil have (f.eks. SMP, PvP, Economy).  
   - **Serverikon** → Upload et .png-ikon (64x64px) for at give din server et personligt look.  
   - **Max Players** → Bestem hvor mange spillere der kan være på samtidig.  
   - **PvP** → Slå spiller-vs-spiller kamp til eller fra.  
   - **Whitelist** → Styr hvem der må join. Du kan tilføje eller fjerne spillere direkte herfra.  

4. **Gem dine ændringer**  
   Når du har foretaget ændringer, klikker du blot på **Gem**.  
   Nogle ændringer kræver en **genstart**, men ofte er det nok at reload'e.

   > Hvis du bliver bedt om at genstarte, kan du gøre det direkte fra panelet — klik blot på "Genstart".

5. **Test ændringerne in-game**  
   Join din server og tjek, om ændringerne ser ud, som du ønsker.  
   Hvis du f.eks. har ændret MOTD eller serverikon, kan du se det allerede i multiplayer-menuen.

### Tips til at få mest ud af Server Manager

- Lav små ændringer ad gangen, så du ved, hvad der gør hvad.
- Brug farvekoder i din MOTD for at gøre den flottere. Du kan finde farvekoder her:  
  *[Farve Koder](https://discord.com/channels/739186031412379738/1083887717508718702)*
- Skift flavour efter sæsoner eller events for at holde serveren spændende.
- Brug whitelist hvis du bygger, tester, eller kun vil have venner på.

### Konklusion

Med Server Manager kan du hurtigt og trygt ændre de vigtigste serverindstillinger — selv hvis du ikke har teknisk erfaring. Det gør det nemt for dig at skabe en server, der passer præcis til det, du og dine spillere ønsker.

Ved at bruge denne guide burde du nu være i stand til at:

 Åbne og navigere i Server Manager  
 Tilpasse serverens udseende og indstillinger  
 Aktivere funktioner som whitelist, PvP og max spillere  
 Gøre serveren mere personlig og indbydende  

God fornøjelse med at tilpasse din server og skabe den bedste spiloplevelse!
